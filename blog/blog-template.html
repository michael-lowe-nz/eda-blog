Grid Based Design

Grid based design is where your whole website can be divided into a grid, where different box elements take up different dimensions of the grid. Therefore a column can be a 3 square wide box on the grid, or 5 squares wide etc. This allows us to easily set up a row and column framework where we can effectively layout websites.
