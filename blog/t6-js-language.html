JavaScript Language

Parentheses ()

Parentheses can be used for:

  • Calling a function

    • doSomething() to use the do something function.

    doSomething(setX, setY) to pass the function some variables that it may need to carry out its function.

  • Order of operations

    • var x = 3 * (5+2)To get javascript to carry out some operation first

  • Method Calls

    • var myString = "Hello"
    myString.substring(0,3)
    To call a method such as substring() on a string

Square Brackets [ ]

Square brackets can be used for:

  • Accesing an Index of Array

    • myArray[0] to access the first item in an array

  • Creating Arrays

    • var myArray = [1,2,3]To create an array containing three integers

Curly Braces { }

Curly braces can be used for:

  • Instantiating Objects

    • myObj = { } to create an object with no properties called myObj

  • A code block after a conditonal

    • if (true){ /** DO THIS **/}Run this block of code if something is true

  • A code block in a loop or while statement

    • while(true){/** DO THIS **/}

    for(char in "dog"){/** DO THIS **/}

Single Quotes ' '

Single quotes can be used for:

  • Denoting a String

    • var myVar = "aString"

  • An apostrophe inside a string

    • var myQuote = "Michael's Blog"

  • Accessing a property of an object

    • anObject["prop"] = "propertyValue"
