Javascript Scope

Scope in javascript is what variables are available to which parts of the program. Let's say we have a program, and we have what is called a global variable. This 'global variable' should be able to be accessed ANYWHERE in the program! Any function or any loop, or whatever, should be able to access this global variable at anytime.

A variable might not have global scope though, it might only be able to be accessed in a particular function. That means that it is locked away inside its own function, other functions don't have the right to access it.
