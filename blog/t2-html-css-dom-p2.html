Grid Based Design

Grid based design is where your whole website can be divided into a grid, where different box elements take up different dimensions of the grid. Therefore a column can be a 3 square wide box on the grid, or 5 squares wide etc. This allows us to easily set up a row and column framework where we can effectively layout websites.

Responsive Webpages

Responsive web design is an essential part of being a web developer. The idea of responsivenss is that webpages aren't just built for one screen size or resolution, they are built to respond to how big the screen size is, which depends on the device that the user is viewing on.

What I find really interesting about responsive design is that it allows for any screen size, even if there aren't any devices with that particular screen size yet. Instead of making sites that look good and function on say a desktop and an iPhone, responsive design ensures that a huge variety of resolutions will work, so the look and functionality of a webpage is future proofed for devices that aren't currently on the market.

Responive design allows a webpage to look good, function well and be easy to use across everything from the smallest smartphones, to big computer monitors, with all the various sizes of tablets in between. And this can all happen with just a few lines of code!

Semantic Structure

Semantic structure is how we lay out our HTML so that both our browser and developers can undestand it. It is the difference between having a whole lot of HTML just in plain text, and having an organised HTML structure with headers and sections and divs.
